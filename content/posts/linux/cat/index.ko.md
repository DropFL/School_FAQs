# cat

Concatenate files and print on the standard output

{{< vs 2 >}}

---
## 용도
파일의 내용을 확인하는 과정에 사용된다.

{{< vs 2 >}}

---
## 옵션

| Useful options | Usage                                                        |
| -------------- | ------------------------------------------------------------ |
| -A             | -vET와 같은 결과가 나온다.                                   |
| -v             | 원래 따로 출력되지 않는 string을 ^ 와 M- 의 형태로 출력한다. |
| -E             | 각 줄의 마지막에 $를 출력한다.                               |
| -T             | ^l의 형태로 TAB을 출력한다.                                  |
