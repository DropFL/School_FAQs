# 인의예지 클러스터가 뭔가요?

인의예지 클러스터는 성대 소프트웨어대학 학생들 + 복수전공 학생들을 위해서 만들어진 클러스터 서버로, 현재 인 서버가 독립되어 있고, 의예지 서버가 nfs로 묶여있는 구조입니다.

자세한 내용은 아래의 공지를 참조해주세요! ([https://cs.skku.edu/news/notice/view/2587](https://cs.skku.edu/news/notice/view/2587))
